# Walker
If you ever lost track of your numerous file graves, decades old, mixed with backups of devices long forgotten and an unknown number of wasteful copies, and you need a tool that can do everything then this tool is not it, but it gives you something close to it: the ability to rediscover your ancient treasures with the power of SQL!

This is a simple tool that traverses a directory and puts file names, extensions, creation and modification dates, hashes, file size into a sqlite database, written in Kotlin.
